Examples

One genome

Here we provide an example of running ExplorePipolin with the Escherichia coli 3-373-03-S1_C2 genome:

  1. Run ExplorePipolin
    2. 
        ~$ explore_pipolin E_coli.fa --out-dir Results --percent-identity 100 --add-colours
  2. Tasks processing
    3.

Multiple genomes

We also wanted to show an example of running ExplorePipolin with multiple genomes. To this purpose we have selected the genomes from the following E. coli strains:

  • Escherichia coli 3-373-03-S1_C2
  • Escherichia coli ATCC 43886
  • Escherichia coli E706
  • Escherichia coli L100
  • Escherichia coli P12b

    2.
